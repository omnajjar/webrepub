## Getting Started

### Prerequisites

In order to run webrepub app locally you need to have [Nodejs](https://nodejs.org/) & `npm` on your machine.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/omnajjar/webrepub.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start development
   ```sh
   npm run dev
   ```
